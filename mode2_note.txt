Note for Summary to Document mode
1. DRM 파일은 사용 할 수 없습니다. 양식 파일은 xlsx 형식만 사용 가능 합니다.
2. Summary 파일의 각 Column 별로 문서가 작성 됩니다.
3. Summary 파일에는 'sheet_name' & 'address' column이 필요하며, 첫번째 행의 입력 값이 파일명으로 이용 됩니다.
4. 작업에 필요한 Summary 파일은 Document to Summary mode를 활용하여, 비교적 간단히 만들 수 있습니다.
5. 이 모드에서는 결과 파일이 서버에 저장 되며, 서버 관리자에 의해 주기적으로 삭제 됩니다. 업로드한 파일은 서버에 저장 되지 않습니다.
6. 서버에 저장 된 결과 ZIP 파일은 사내 망 사용자에게 노출 될 수 있습니다. 결과 ZIP 파일의 보안을 위해, 암호를 적용 할 수 있습니다.
